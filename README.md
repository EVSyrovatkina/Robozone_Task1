# Цифровой двойник сортировочного центра (СЦ) Ozon Tech

Имитационная модель дискретно-событийного моделирования потоков СЦ для Задачи 1.

## Инструкция по запуску

### Способ 1: Запуск в облаке через Google Colab (Рекомендуемый)
1. Откройте пустой блокнот на [Google Colab](https://colab.new).
2. В первой ячейке установите библиотеку симуляции, выполнив команду: `!pip install simpy`
3. Создайте новую ячейку, скопируйте туда содержимое файла `simulation_model.py` из этого репозитория и нажмите **Ctrl + Enter**.

### Способ 2: Локальный запуск на ПК
1. Установите библиотеку симуляции через терминал: `pip install simpy`
2. Запустите скрипт: `python simulation_model.py`
